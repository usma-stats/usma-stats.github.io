MA206X Course Syllabus

AY26-2: Probability and Statistics

Author

CPT Jonathan Day

1 Course Information

Course: MA206X - Probability and Statistics Credits: 3.0 Textbook: Probability and Statistics for Engineering and the Sciences, 9th Ed., by Jay Devore Platform: Cengage WebAssign (for homework)

2 Course Description

This course introduces you to the foundational principles of probability and statistics, emphasizing data literacy and inference. It begins with Block I, covering data types, visualization, and basic probability rules including counting and the behavior of random variables. Block II builds on this by exploring discrete and continuous distributions, the Central Limit Theorem, and tools for one-sample inference such as confidence intervals and hypothesis testing for proportions and means. Finally, Block III develops cadets’ ability to analyze relationships between variables through two-sample tests, linear regression, ANOVA, and goodness-of-fit testing. By the end of the course, cadets will be equipped to make sound, data-driven decisions grounded in statistical reasoning.

3 Learning Objectives

By completing MA206X, you will be able to:

  1. Apply the Six Step Method of statistical investigation
  2. Classify and visualize data appropriately
  3. Calculate and interpret descriptive statistics
  4. Apply fundamental probability rules and counting principles
  5. Work with discrete and continuous probability distributions
  6. Construct and interpret confidence intervals
  7. Conduct hypothesis tests for means and proportions
  8. Perform linear regression and ANOVA
  9. Use R for statistical analysis and visualization
  10. Communicate statistical findings effectively

6 Course Schedule

6.1 Block I: Data and Randomness (Lessons 1-16)

Foundational concepts in data types, sampling, exploratory data analysis, probability theory, and discrete/continuous distributions.

Topics: Data types, sampling methods, bias, measures of location and spread, probability axioms, conditional probability, counting, discrete random variables, binomial/Poisson distributions, continuous random variables, normal/exponential distributions.

Assessment: WPR I (Lesson 16)

6.2 Block II: Inference (Lessons 17-27)

Statistical inference for single populations and comparisons between two populations.

Topics: Central Limit Theorem, confidence intervals, hypothesis testing framework, one-sample tests (t-test, z-test), two-sample tests, paired data, Type I/II errors, statistical power.

Assessment: WPR II (Lesson 27)

6.3 Block III: Regression Modeling (Lessons 28-40)

Modeling relationships between variables and analysis of variance.

Topics: Simple linear regression, multiple regression, categorical predictors, interaction terms, model diagnostics, ANOVA (one-way and two-way), multiple comparisons, project work.

Assessment: Tech Report (Lesson 36), Project Presentation (Lessons 39-40), TEE

7 Course Project

Throughout this course you will be executing a statistical investigation into data hosted inside the Vantage enterprise on actual data from an Army unit. You will:

  1. Select a dataset from Army Vantage
  2. Formulate research questions
  3. Conduct exploratory data analysis
  4. Perform appropriate statistical tests
  5. Build regression models
  6. Present findings in a technical report and presentation

Deliverables:

  • EDA Assignment (Lesson 5)
  • Technical Report (Lesson 36)
  • Peer Review (Lesson 34)
  • IPR Presentation (Lesson 37)
  • Final Presentation (Lessons 39-40)
