0.1 Learning Objectives

By the end of this lesson, you will be able to:

  1. Distinguish populations, samples, and processes in studies
  2. Classify variables as categorical or quantitative
  3. Identify parameters vs. statistics in context

Reading: Devore 1.1

1 Introduction

Statistical thinking begins with understanding what we’re measuring and how we collect that information. Before we can analyze data, we must clearly define:

  • What entities we’re studying (observational units)
  • What characteristics we’re measuring (variables)
  • Whether we’re studying everyone or just a subset (population vs. sample)

2 Populations, Samples, and Processes

2.1 Key Definitions

Fundamental Concepts

Population: The entire collection of individuals or objects about which information is desired.

Sample: A subset of the population selected for study.

Parameter: A numerical characteristic of a population (usually unknown).

Statistic: A numerical characteristic of a sample (computed from data).

2.2 Why Study Samples?

In most real-world scenarios, studying an entire population is:

  • Impractical - Too time-consuming or expensive
  • Impossible - Population is infinite or inaccessible
  • Unnecessary - A well-chosen sample provides reliable information

The key challenge: How do we ensure our sample accurately represents the population?

2.3 Example 1.1: Population vs. Sample

Scenario: You want to know the average height of all cadets in the Corps.

  • Population: All 4,400 cadets at USMA
  • Sample: 200 randomly selected cadets
  • Parameter: \(\mu\) = true average height of all cadets (unknown)
  • Statistic: \(\bar{x}\) = average height of the 200 sampled cadets (computable)

If we calculate \(\bar{x} = 69.5\) inches from our sample, we estimate that \(\mu \approx 69.5\) inches.

3 Types of Variables

Variables are characteristics that can differ from one observational unit to another. Understanding variable types determines which statistical methods we can use.

3.1 Categorical Variables

Categorical (qualitative) variables place individuals into groups or categories.

3.1.1 Subtypes:

  1. Nominal: Categories with no natural ordering
    • Examples: Color (red, blue, green), Military Occupational Specialty (11A, 35D, 12A)
  2. Ordinal: Categories with a meaningful order
    • Examples: Class rank (4th, 3rd, 2nd, 1st class), Likert scale (Strongly Disagree → Strongly Agree)
  3. Binary: Only two possible categories
    • Examples: Pass/Fail, Yes/No, True/False

3.2 Quantitative Variables

Quantitative (numerical) variables take numerical values where arithmetic operations make sense.

3.2.1 Subtypes:

  1. Discrete: Countable values (often integers)
    • Examples: Number of pushups, number of siblings, count of vehicles
  2. Continuous: Can take any value in an interval
    • Examples: Height, weight, temperature, time

3.3 Quick Classification Guide

Question If Yes If No
Can you perform arithmetic? Quantitative Categorical
Are values countable? Discrete Continuous
Is there a natural order? Ordinal Nominal

Let’s classify variables from a cadet fitness study:

5 Parameters vs. Statistics: Notation Matters

Statistical inference involves using sample statistics to make claims about population parameters. We use different notation for each:

Measure Population Sample
Mean $\mu$ (mu) $\bar{x}$ (x-bar)
Proportion $\pi$ or $p$ $\hat{p}$ (p-hat)
Standard Deviation $\sigma$ (sigma) $s$
Variance $\sigma^2$ $s^2$
Correlation $\rho$ (rho) $r$
Size $N$ $n$
Common Mistake

Don’t confuse population proportion \(p\) with p-value! Context matters.

  • When discussing “true proportion of successes” → population parameter \(p\) or \(\pi\)
  • When discussing “probability of observing data if null hypothesis is true” → p-value

5.1 Example 1.3: Parameters and Statistics

Scenario: The Army wants to know what percentage of all soldiers prefer virtual training to in-person training.

  • Population: All active-duty soldiers
  • Parameter of interest: \(\pi\) = proportion of ALL soldiers who prefer virtual training
  • Sample: 1,000 randomly selected soldiers
  • Statistic: \(\hat{p} = 0.62\) (62% of sampled soldiers prefer virtual)

Inference: We use \(\hat{p} = 0.62\) to estimate that \(\pi \approx 0.62\).

Later in the course, we’ll learn how to quantify our uncertainty about this estimate using confidence intervals.
