Deriving the OLS Estimators

How does Ordinary Least Squares solve for the slope and intercept?

1. The Setup

We model the relationship between a response $y$ and predictor $x$ as a linear function:

$$\hat{y}_i = \hat{\beta}_0 + \hat{\beta}_1 x_i$$

For each observation $i$, the residual is the vertical distance between the observed value and the predicted value:

$$e_i = y_i - \hat{y}_i = y_i - \hat{\beta}_0 - \hat{\beta}_1 x_i$$

The sum of squared errors (SSE) measures total prediction error across all $n$ observations:

$$\text{SSE} = \sum_{i=1}^{n} e_i^2 = \sum_{i=1}^{n} \bigl(y_i - \hat{\beta}_0 - \hat{\beta}_1 x_i\bigr)^2$$

Our goal: find the values of $\hat{\beta}_0$ and $\hat{\beta}_1$ that make SSE as small as possible.

2. Solving for $\hat{\beta}_0$

Step 2a

Take the partial derivative of SSE with respect to $\hat{\beta}_0$ and set it equal to zero:

$$\frac{\partial}{\partial \hat{\beta}_0} \sum_{i=1}^{n} \bigl(y_i - \hat{\beta}_0 - \hat{\beta}_1 x_i\bigr)^2 = 0$$

Applying the chain rule:

$$-2 \sum_{i=1}^{n} \bigl(y_i - \hat{\beta}_0 - \hat{\beta}_1 x_i\bigr) = 0$$

Step 2b

Divide both sides by $-2$ and distribute the sum:

$$\sum_{i=1}^{n} y_i - n\hat{\beta}_0 - \hat{\beta}_1 \sum_{i=1}^{n} x_i = 0$$

Step 2c

Divide everything by $n$ and recognize that $\frac{1}{n}\sum y_i = \bar{y}$ and $\frac{1}{n}\sum x_i = \bar{x}$:

$$\bar{y} - \hat{\beta}_0 - \hat{\beta}_1 \bar{x} = 0$$

Solving for $\hat{\beta}_0$:

$$\boxed{\hat{\beta}_0 = \bar{y} - \hat{\beta}_1 \bar{x}}$$

This tells us the OLS line always passes through the point $(\bar{x}, \bar{y})$ — which you can verify in the interactive plot above by toggling "Show x̄, ȳ lines."

3. Solving for $\hat{\beta}_1$

Step 3a

Take the partial derivative of SSE with respect to $\hat{\beta}_1$ and set it equal to zero:

$$\frac{\partial}{\partial \hat{\beta}_1} \sum_{i=1}^{n} \bigl(y_i - \hat{\beta}_0 - \hat{\beta}_1 x_i\bigr)^2 = 0$$

Applying the chain rule (note the extra $-x_i$ factor):

$$-2 \sum_{i=1}^{n} x_i\bigl(y_i - \hat{\beta}_0 - \hat{\beta}_1 x_i\bigr) = 0$$

Step 3b

Divide by $-2$ and expand:

$$\sum_{i=1}^{n} x_i y_i - \hat{\beta}_0 \sum_{i=1}^{n} x_i - \hat{\beta}_1 \sum_{i=1}^{n} x_i^2 = 0$$

Step 3c

Substitute $\hat{\beta}_0 = \bar{y} - \hat{\beta}_1 \bar{x}$ from Step 2:

$$\sum_{i=1}^{n} x_i y_i - (\bar{y} - \hat{\beta}_1 \bar{x})\sum_{i=1}^{n} x_i - \hat{\beta}_1 \sum_{i=1}^{n} x_i^2 = 0$$

Since $\sum x_i = n\bar{x}$, expand and collect terms:

$$\sum_{i=1}^{n} x_i y_i - n\bar{x}\bar{y} + \hat{\beta}_1 n\bar{x}^2 - \hat{\beta}_1 \sum_{i=1}^{n} x_i^2 = 0$$

Step 3d

Factor out $\hat{\beta}_1$ and solve:

$$\hat{\beta}_1 \left(\sum_{i=1}^{n} x_i^2 - n\bar{x}^2\right) = \sum_{i=1}^{n} x_i y_i - n\bar{x}\bar{y}$$

Recognizing that $\sum x_i^2 - n\bar{x}^2 = \sum(x_i - \bar{x})^2 = SS_{xx}$ and $\sum x_i y_i - n\bar{x}\bar{y} = \sum(x_i - \bar{x})(y_i - \bar{y}) = SS_{xy}$:

$$\boxed{\hat{\beta}_1 = \frac{SS_{xy}}{SS_{xx}} = \frac{\displaystyle\sum_{i=1}^{n}(x_i - \bar{x})(y_i - \bar{y})}{\displaystyle\sum_{i=1}^{n}(x_i - \bar{x})^2}}$$

4. Summary

The OLS estimators for simple linear regression $\hat{y} = \hat{\beta}_0 + \hat{\beta}_1 x$ are:

$$\hat{\beta}_1 = \frac{\sum_{i=1}^{n}(x_i - \bar{x})(y_i - \bar{y})}{\sum_{i=1}^{n}(x_i - \bar{x})^2} = \frac{SS_{xy}}{SS_{xx}}$$ $$\hat{\beta}_0 = \bar{y} - \hat{\beta}_1\bar{x}$$

These are the unique values that minimize $\text{SSE} = \sum(y_i - \hat{\beta}_0 - \hat{\beta}_1 x_i)^2$. No other choice of slope and intercept can produce a smaller SSE — which is exactly what you can verify by playing with the "Fit Your Own" mode above.

The second-order conditions (the Hessian matrix of SSE is positive definite whenever $SS_{xx} > 0$, i.e., not all $x_i$ are identical) confirm this is a true minimum, not a maximum or saddle point.
